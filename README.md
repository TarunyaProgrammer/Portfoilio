> **"A portfolio you don’t touch — you command."**

Airfolio is a futuristic, **gesture-controlled** developer portfolio built for **Tarunya Kesharwani**. It completely reimagines web navigation by replacing the mouse scroll with mid-air hand gestures, powered by AI.

---

## ✨ Features

- **🖐️ Touchless Navigation**: Scroll the entire website using your index finger and thumb via webcam.
- **🔮 Physics-Based Scrolling**: Custom inertia engine for buttery smooth scroll performance.
- **🎥 Cinematic Motion**: Powered by **GSAP ScrollTrigger** for award-winning scroll animations.
- **🌑 Cyber-Clean Aesthetic**: Deep Space Blue (`#05080F`) & Neon Cyan (`#00F5FF`) dark glassmorphism theme.
- **🔒 Pinch-to-Scroll Safety**: Prevents accidental scrolling with a smart pinch-detection lock.
- **⚡ Zero Frameworks**: Built with pure Vanilla JS for maximum performance.

---

## 🛠️ Tech Stack

- **Core**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **AI / Vision**: [MediaPipe Hands](https://developers.google.com/mediapipe) (Google)
- **Animation**: [GSAP Core + ScrollTrigger](https://gsap.com)
- **Design**: CSS Variables, Glassmorphism, CSS Grid

---

## 🚀 How It Works

1.  **Grant Camera Access**: The site needs your webcam to track your hand (processed locally, privacy-first).
2.  **Pinch Your Fingers**: Pinch your **Index Finger** and **Thumb** together to "grab" the scroll handle.
3.  **Move Up/Down**:
    - Move hand **UP** to scroll **DOWN**.
    - Move hand **DOWN** to scroll **UP**.
4.  **Release to Stop**: Open your fingers to stop scrolling instantly.
